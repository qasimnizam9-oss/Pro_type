# Pro_type
ProType is a high-performance, professional-grade typing speed analyzer built with Python and Tkinter. It provides real-time feedback, persistent statistics tracking, and a modern, distraction-free interface designed for serious typists and developers looking to improve their KPM (Keys Per Minute) and accuracy.
Real-Time Analytics: Live WPM (Words Per Minute) and accuracy calculation as you type.

Persistent Progress: Saves your lifetime "All-Time High" and "Low" scores using local JSON storage.

Visual Feedback: Dynamic UI color shifting (Green/Red) to indicate accuracy and errors instantly.

Continuous Training: Automatically cycles through a diverse library of technical and literary paragraphs.

Clean Sidebar Interface: Integrated controls for resetting tests, clearing lifetime data, and safe exiting.
